A Missão

O GCP cobra se você descuida.
O Guardian cuida por você.

O Google Cloud tem um Free Tier generoso — VM e2-micro, 5GB de Storage, 2M de Cloud Run requests por mês. Ótimo para projetos pessoais e startups early-stage. O problema? Um vazamento de memória, um loop infinito ou um bucket público esquecido pode gerar centenas de dólares em cobrança em questão de horas.

O GCP Guardian nasceu de uma necessidade real: rodar projetos profissionais na nuvem sem ansiedade financeira. Ele age como um guardrail ativo — coletando métricas a cada 15 minutos, projetando o consumo até o fim do período e executando ações de contenção automáticas quando os limites são ameaçados.

Sem surpresas. Sem cobranças inesperadas. Zero custo de hosting.

Como funciona

Três camadas de proteção.
Uma missão: não pagar nada.

→
1
Coleta de Métricas

A cada 15 minutos, o backend .NET consulta a Cloud Monitoring API do GCP e grava as métricas de consumo no DuckDB local — sem custo de banco de dados.

→
2
Análise & Alertas

O Guardrail Engine avalia os thresholds (75%, 90%, 98%) e envia alertas via Telegram com projeção de quando o limite será atingido.

3
Kill-Switch Automático

Em emergência (98%), o sistema executa ações programáticas via API: limita instâncias do Cloud Run, para serviços e evita cobranças.

Funcionalidades

Tudo que você precisa para
dormir tranquilo na nuvem.

📊
Dashboard em Tempo Real

Interface web PHP + Vue.js com gráficos ECharts mostrando o consumo atual, barras de progresso por serviço e projeção até fim do mês.

🔔
Alertas Telegram

Notificações instantâneas quando qualquer quota ultrapassa 75%, 90% ou 98%. Com fail-safe: se o Telegram falhar, o sistema continua funcionando.

🛡️
Guardrail Engine

Motor de regras com thresholds configuráveis. Ações automáticas: notificação → limitação → kill-switch. Cada nível é independente.

🐦
DuckDB Local

Banco analítico embutido — sem servidor, sem custo adicional. Histórico de métricas com suporte a sync para GCS em formato Parquet.

🚀
CI/CD Automático

48 testes TDD com GitHub Actions gratuito. Deploy via SSH com smoke test. Dependabot para atualização automática de dependências.

💰
100% Gratuito

Roda na VM e2-micro (Always Free). Nginx + PHP + .NET em containers Docker na mesma máquina. Terraform para provisionar com um comando.

Thresholds & Ações

Três níveis de alerta.
Ações progressivas e automáticas.

Consumo Nível Ação automática Notificação
< 75% ✓ OK Nenhuma — monitoramento contínuo Silencioso
≥ 75% ⚠ Warning Registra alerta + envia Telegram 📱 Telegram imediato
≥ 90% 🔴 Critical Cloud Run limitado a 1 instância via API 📱 Telegram urgente
≥ 98% 🚨 Emergency Kill-switch: Cloud Run parado + serviços suspensos 📱 Telegram crítico + log de auditoria
